# 🏋️ FitBuddy – AI Fitness Plan Generator

FitBuddy is a web-based AI application that generates personalized 7-day workout plans using Google Gemini AI.

## 🚀 Live Demo
🔗 https://fitbuddy-ai-1.onrender.com

## ✨ Features

- 🤖 AI-generated 7-day workout plan  
- 🥗 Smart nutrition tips  
- 🛌 Recovery recommendations  
- 🔁 Feedback-based plan regeneration  
- 💾 SQLite database storage  
- 📊 Clean HTML UI  
- 📝 Logging & error handling  

## 🧠 Tech Stack

- FastAPI  
- Google Gemini AI  
- SQLite  
- HTML/CSS  
- Render (Deployment)

## 📦 Installation

```bash
git clone https://github.com/amar2004-ak/fitbuddy-ai.git
cd fitbuddy-ai
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
